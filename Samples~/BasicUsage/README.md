# Basic Usage Sample

This sample demonstrates basic usage of Rula.Persistence.Unity.

The example shows:

- Creating a SaveManager using Unity factory.
- Registering saveable data.
- Saving player state.
- Loading player state.

## Usage

Import this sample through Unity Package Manager.

Add `PersistenceExample` component to a GameObject and assign UI references.

Press:
- Increase button to modify data.
- Save button to store data.

The save file is stored using Unity persistent data path.
